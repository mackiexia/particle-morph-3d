An immersive 3D particle interactive toy that builds sculptural art forms with text symbols.

一个沉浸式3D粒子交互玩具，用文字符号构建立体艺术模型。

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1DYBfzLzVlT04o7-MoWYhfKEJMz3FjUdi

## Features
7 3D Models
Möbius Strip · Pentagram · Klein Bottle · Chinese Incense Burner · Chinese Phoenix Crown · DNA Double Helix · Relief
8 Color Themes
Dunhuang · Space · Klein Blue · Morandi · Mondrian · Rococo · Matisse · Memphis

### Interactions

- Drag to rotate
- Scroll to zoom
- Custom particle symbols (supports Chinese, English, Emoji)
- Real-time particle density control (500-2000)

## 特性
7种立体模型
莫比乌斯环 · 五角星 · 克莱因瓶 · 中式香炉 · 中式凤冠 · DNA双螺旋 · 浮雕
8种艺术配色
敦煌 · 太空 · 克莱因蓝 · 莫兰迪 · 蒙德里安 · 洛可可 · 马蒂斯 · 孟菲斯

### 自由交互

- 拖拽旋转
- 滚轮缩放
- 自定义粒子符号（中/英/Emoji）
- 实时调节粒子密度（500-2000）

## Run Locally / 本地运行

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
